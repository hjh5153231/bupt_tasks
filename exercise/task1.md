
# Task 1

## 题目: 简易计算器


## 目标
- 了解CPP编程流程


### 功能
- 用户通过命令行输入计算式, 回车后打印计算结果, 并等待用户输入下一个计算式
- 需要至少支持`加/减/乘/除`四种运算
- 支持小数运算
- 支持括号

### 要求
- 使用 CMake 创建工程
