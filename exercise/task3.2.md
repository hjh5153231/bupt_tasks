
# Task 3.2

## 题目: 基于 OpenAL 的 PCM 播放器


## 目标
- 熟悉 CPP 编程流程
- 了解 CMake 配置依赖流程
- 了解音频播放基本原理


### 内容
- 通过ffmpeg命令行将从视频/音频文件中提取PCM音频
- 基于 OpenAL, 编程实现对 PCM 格式的播放

### 要求
- 使用 CMake 创建工程
- 除了`OpenAL`外, 不引入其他第三方库
- 除了正常播放外, 提供`暂停`, `重新播放`, `调节音量`功能
